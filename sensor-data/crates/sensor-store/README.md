# Sensor Store

ORM-like crate to interact with the sensor database.
Useful to retrieve sensor fields, units, etc.
This crate defines the mapping from sensor quantities to the base unit/units and vica versa.
Quantities being the kind of signal being measured, for example, consumption, electric potential, rainfall, etc.
